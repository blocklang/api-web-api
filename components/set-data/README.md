# `setData(key: string, value: any)`

`setData` 函数用于修改页面数据，并重新渲染页面中关联的部件。

## 参数说明

| 参数名  | 类型   | 必填 | 描述                                                 | 最低版本 |
| ------- | ------ | ---- | ---------------------------------------------------- | -------- |
| `key`   | string | 是   | key 是 json path，如 `a.b`                           |          |
| `value` | any    | 是   | 值，可以整个对象或数组，但推荐修改对象或数组的子元素 |          |
