# `getData(key: string): any`

`getData` 函数用于获取页面数据。

## 参数说明

| 参数名 | 类型   | 必填 | 说明                       | 最低版本 |
| ------ | ------ | ---- | -------------------------- | -------- |
| `key`  | string | 是   | key 是 json path，如 `a.b` |          |

## 返回值

| 属性    | 类型 | 说明                   |
| ------- | ---- | ---------------------- |
| `value` | any  | 值，可以整个对象或数组 |
